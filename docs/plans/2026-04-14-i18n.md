# i18n Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Ajouter l'internationalisation complète avec next-intl sur 12 locales (fr, en, es, de, it, pt, nl, pt-BR, es-MX, ja, zh, ko) avec détection automatique via le navigateur et routes préfixées `/[locale]/`.

**Architecture:** next-intl avec middleware pour la détection de langue. Toutes les pages publiques et authentifiées passent sous `src/app/[locale]/`. Les routes API (`/api/...`) restent hors du segment `[locale]`. Fallback : `en`.

**Tech Stack:** next-intl ^4, Next.js 16 App Router, TypeScript strict

### Task 2 : Créer la config i18n (routing + request + middleware)

**Files:**
- Create: `src/i18n/routing.ts`
- Create: `src/i18n/request.ts`
- Create: `src/middleware.ts`

**Step 1 : Créer `src/i18n/routing.ts`**

```ts
import { defineRouting } from "next-intl/routing";

export const routing = defineRouting({
  locales: ["en", "fr", "es", "de", "it", "pt", "nl", "pt-BR", "es-MX", "ja", "zh", "ko"],
  defaultLocale: "en",
  localeDetection: true,
});
```

**Step 2 : Créer `src/i18n/request.ts`**

```ts
import { getRequestConfig } from "next-intl/server";
import { routing } from "./routing";

export default getRequestConfig(async ({ requestLocale }) => {
  let locale = await requestLocale;

  if (!locale || !routing.locales.includes(locale as never)) {
    locale = routing.defaultLocale;
  }

  return {
    locale,
    messages: (await import(`../../messages/${locale}.json`)).default,
  };
});
```

**Step 3 : Créer `src/middleware.ts`**

```ts
import createMiddleware from "next-intl/middleware";
import { routing } from "./i18n/routing";

export default createMiddleware(routing);

export const config = {
  matcher: [
    "/((?!api|_next|_vercel|.*\\..*).*)",
  ],
};
```

**Step 4 : Vérifier le build**

```bash
npm run build
```

**Step 5 : Commit**

```bash
git add src/i18n/ src/middleware.ts
git commit -m "feat(i18n): config next-intl routing, request et middleware"
```

Ce layout gère next-intl, les métadonnées SEO complètes, Footer et ReviewBadge :

```tsx
import type { Metadata } from "next";
import { NextIntlClientProvider } from "next-intl";
import { getMessages, getLocale } from "next-intl/server";
import { notFound } from "next/navigation";
import { routing } from "@/i18n/routing";
import Providers from "@/components/Providers";
import { ReviewBadge } from "@/components/ReviewBadge";
import { Footer } from "@/components/Footer";
import { BASE_URL } from "@/lib/config";

export async function generateMetadata({ params }: { params: Promise<{ locale: string }> }): Promise<Metadata> {
  const { locale } = await params;
  return {
    title: {
      default: "QRaft — Free QR Code Generator",
      template: "%s | QRaft",
    },
    description: "Create customized QR codes for free in seconds.",
    metadataBase: new URL(BASE_URL),
    alternates: {
      canonical: `${BASE_URL}/${locale}`,
      languages: Object.fromEntries(
        routing.locales.map((l) => [l, `${BASE_URL}/${l}`])
      ),
    },
  };
}

export function generateStaticParams() {
  return routing.locales.map((locale) => ({ locale }));
}

export default async function LocaleLayout({
  children,
  params,
}: {
  children: React.ReactNode;
  params: Promise<{ locale: string }>;
}) {
  const { locale } = await params;

  if (!routing.locales.includes(locale as never)) {
    notFound();
  }

  const messages = await getMessages();

  return (
    <html lang={locale}>
      <body>
        <NextIntlClientProvider messages={messages}>
          <Providers>
            {children}
          </Providers>
          <Footer />
          <ReviewBadge />
        </NextIntlClientProvider>
      </body>
    </html>
  );
}
```

**Step 1 : Mettre à jour le sitemap pour inclure toutes les locales**

```ts
import { routing } from "@/i18n/routing";
import { BASE_URL } from "@/lib/config";
import type { MetadataRoute } from "next";

export default function sitemap(): MetadataRoute.Sitemap {
  const locales = routing.locales;
  const pages = ["", "/login", "/register"];

  return locales.flatMap((locale) =>
    pages.map((page) => ({
      url: `${BASE_URL}/${locale}${page}`,
      lastModified: new Date(),
      changeFrequency: "monthly" as const,
      priority: page === "" ? 1 : 0.8,
    }))
  );
}
```

**Step 2 : Mettre à jour robots.ts**

import { BASE_URL } from "@/lib/config";

export default function robots(): MetadataRoute.Robots {

}
