## 为什么使用 WiFi 二维码？

分享 WiFi 密码仍然是日常生活中最令人头疼的事情之一。您一定遇到过：给客人、餐厅顾客或活动参与者逐字拼读 20 位密码。WiFi 二维码一步即可解决这个问题。

当有人扫描您的 WiFi 二维码时，手机会**自动**连接到网络。无需手动输入，不会打错字，不会混淆大小写。

**最常见的使用场景：**

- 提供免费 WiFi 的餐厅和咖啡馆
- 酒店和民宿用于客人入住
- 办公室和共享办公空间
- 活动和会议
- 商店和候诊室

## 如何生成您的 WiFi 二维码

### 第一步：收集您的网络信息

您需要三样东西：

- **网络名称（SSID）**：必须与路由器上显示的完全一致。注意大小写和空格。
- **安全类型**：WPA/WPA2（最常见）、WEP（旧式）或 Open（无密码）。
- **WiFi 密码**：原样输入，区分大小写。

<InfoBox type="tip">
要查找您的 SSID 和安全类型，请打开手机的 WiFi 设置，查看已连接网络的详细信息。
</InfoBox>

### 第二步：生成二维码

前往 [QRaft WiFi 生成器](/qr-code-generator/wifi) 并填写字段：

1. 输入您的网络名称（SSID）
2. 选择安全类型
3. 输入密码
4. 如适用，勾选"隐藏网络"

二维码会在预览区实时生成。

### 第三步：下载并打印

选择适合您的格式：

- **PNG**：适用于网页和标准打印
- **SVG**：矢量格式，缩放不失真
- **PDF**：可直接打印

### 第四步：战略性摆放

摆放位置决定使用效果：

- **餐厅**：每张桌子上，菜单旁
- **酒店**：前台和每个房间
- **办公室**：会议室和公共区域
- **活动**：入口处和胸牌上

## 高效 WiFi 二维码的小贴士

1. **打印足够大**：最小 3 厘米 × 3 厘米以确保可靠扫描
2. **良好对比度**：浅色背景上的深色二维码
3. **打印前测试**：务必用另一部手机扫描您的二维码
4. **更改密码后要更新**：静态二维码打印后无法修改
5. **添加提示文字**："扫码连接 WiFi"可以帮助不太熟悉二维码的人
