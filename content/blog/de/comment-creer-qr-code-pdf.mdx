## Warum ein PDF mit einem QR-Code verknuepfen?

PDF bleibt das universellste Dokumentformat. Einen QR-Code mit einem PDF zu verknuepfen, bietet sofortigen Zugang zu:

- **Restaurantkarten**: Vollversion mit Fotos
- **Verkaufsbroschueren**: detaillierter Produktkatalog
- **Bedienungsanleitungen**: Geraeteinstruktionen
- **Preislisten**: aktuelle Preise
- **Lebenslaeaufe**: Vollversion Ihres beruflichen Werdegangs
- **AGB**: rechtliche Dokumente

Der Vorteil: der QR-Code ist klein (ein Aufkleber genuegt), bietet aber Zugang zu einem mehrseitigen Dokument.

## Voraussetzung: Hosten Sie Ihr PDF online

Der QR-Code verweist auf eine URL. Ihr PDF muss ueber einen oeffentlichen Link erreichbar sein.

### Google Drive (empfohlen)
1. Laden Sie Ihr PDF auf Google Drive hoch
2. Rechtsklick → Freigeben → „Jeder mit dem Link"
3. Link kopieren

<InfoBox type="tip">
Fuer einen Direktlink (der das PDF ohne die Google-Drive-Oberflaeche oeffnet), ersetzen Sie `/file/d/ID/view` durch `/uc?export=download&id=ID` in der URL.
</InfoBox>

### Dropbox
1. Laden Sie das PDF hoch
2. Klicken Sie auf „Link kopieren"
3. Ersetzen Sie `dl=0` durch `dl=1` fuer einen Direktdownload

### Ihre Webseite
Laden Sie das PDF auf Ihr Hosting hoch und verwenden Sie die direkte URL (z.B. `https://meineseite.de/docs/broschuere.pdf`).

## QR-Code generieren

1. Gehen Sie zum [useqraft PDF-Generator](/qr-code-generator/pdf)
2. Fuegen Sie Ihre PDF-URL ein
3. Ueberpruefen Sie die Vorschau
4. Herunterladen als PNG, SVG oder PDF

## Tipp: Aktualisieren ohne Nachdruck

Wenn Sie die **gleiche URL** beibehalten und die PDF-Datei an dieser Adresse ersetzen, verweist der QR-Code automatisch auf die neue Version.

Mit einem useqraft-Pro-Konto koennen Sie auch einen **dynamischen** QR-Code erstellen, dessen Ziel-URL jederzeit geaendert werden kann.

## Best Practices

1. **PDF-Groesse optimieren**: komprimieren Sie es fuer schnelles Laden auf dem Handy
2. **Responsive gestalten**: stellen Sie sicher, dass der Text auf dem Handybildschirm lesbar ist
3. **Link testen**: scannen Sie den QR-Code von einem anderen Geraet, bevor Sie ihn verteilen
4. **Handlungsaufforderung hinzufuegen**: „Scannen Sie, um unseren Katalog anzusehen"
