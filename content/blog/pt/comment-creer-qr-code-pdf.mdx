## Porquê Associar um PDF a um Código QR?

O PDF continua a ser o formato de documento mais universal. Associar um código QR a um PDF dá acesso instantâneo a:

- **Menus de restaurante**: versão completa com fotografias
- **Brochuras comerciais**: catálogo detalhado de produtos
- **Manuais de utilizador**: instruções de dispositivos
- **Tabelas de preços**: preços atualizados
- **Currículos**: versão completa do seu percurso profissional
- **Termos e condições**: documentos legais

A vantagem: o código QR é pequeno (basta um autocolante), mas dá acesso a um documento de várias páginas.

## Pré-requisito: Alojar o Seu PDF Online

O código QR aponta para um URL. O seu PDF deve estar acessível através de uma ligação pública.

### Google Drive (Recomendado)
1. Carregue o seu PDF para o Google Drive
2. Clique direito → Partilhar → "Qualquer pessoa com a ligação"
3. Copie a ligação

<InfoBox type="tip">
Para uma ligação direta (que abre o PDF sem a interface do Google Drive), substitua `/file/d/ID/view` por `/uc?export=download&id=ID` no URL.
</InfoBox>

### Dropbox
1. Carregue o PDF
2. Clique em "Copiar ligação"
3. Substitua `dl=0` por `dl=1` para uma transferência direta

### O Seu Website
Carregue o PDF para o seu alojamento e utilize o URL direto (ex: `https://meusite.com/docs/brochura.pdf`).

## Gere o Código QR

1. Aceda ao [gerador de PDF do QRaft](/qr-code-generator/pdf)
2. Cole o URL do seu PDF
3. Verifique a pré-visualização
4. Transfira em PNG, SVG ou PDF

## Dica: Atualize Sem Reimprimir

Se mantiver o **mesmo URL** e substituir o ficheiro PDF nesse endereço, o código QR apontará automaticamente para a nova versão.

Com uma conta QRaft Pro, também pode criar um código QR **dinâmico** cujo URL de destino pode ser alterado a qualquer momento.

## Boas Práticas

1. **Otimize o tamanho do PDF**: comprima-o para carregamento rápido no telemóvel
2. **Torne-o responsivo**: garanta que o texto é legível no ecrã de um telemóvel
3. **Teste a ligação**: leia o código QR a partir de outro dispositivo antes de distribuir
4. **Adicione uma chamada à ação**: "Leia para ver o nosso catálogo"
