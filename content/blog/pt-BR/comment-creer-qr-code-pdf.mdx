## Por Que Vincular um PDF a um QR Code?

O PDF continua sendo o formato de documento mais universal. Vincular um QR code a um PDF dá acesso instantâneo a:

- **Cardápios de restaurantes**: versão completa com fotos
- **Folhetos comerciais**: catálogo detalhado de produtos
- **Manuais de uso**: instruções de dispositivos
- **Tabelas de preços**: valores atualizados
- **Currículos**: versão completa do seu histórico profissional
- **Termos e condições**: documentos legais

A vantagem: o QR code é pequeno (um adesivo basta) mas dá acesso a um documento de várias páginas.

## Pré-requisito: Hospede Seu PDF Online

O QR code aponta para uma URL. Seu PDF precisa estar acessível via um link público.

### Google Drive (Recomendado)
1. Faça upload do seu PDF no Google Drive
2. Clique com o botão direito → Compartilhar → "Qualquer pessoa com o link"
3. Copie o link

<InfoBox type="tip">
Para um link direto (que abre o PDF sem a interface do Google Drive), substitua `/file/d/ID/view` por `/uc?export=download&id=ID` na URL.
</InfoBox>

### Dropbox
1. Faça upload do PDF
2. Clique em "Copiar link"
3. Substitua `dl=0` por `dl=1` para download direto

### Seu Site
Faça upload do PDF na sua hospedagem e use a URL direta (ex.: `https://meusite.com/docs/folheto.pdf`).

## Gere o QR Code

## Dica: Atualize Sem Reimprimir

Se você mantiver a **mesma URL** e substituir o arquivo PDF nesse endereço, o QR code apontará automaticamente para a nova versão.

Com uma conta QRaft Pro, você também pode criar um QR code **dinâmico** cuja URL de destino pode ser alterada a qualquer momento.

## Boas Práticas

1. **Otimize o tamanho do PDF**: comprima-o para carregamento rápido no celular
2. **Torne-o responsivo**: garanta que o texto seja legível na tela do celular
3. **Teste o link**: escaneie o QR code de outro dispositivo antes de distribuir
4. **Adicione uma chamada para ação**: "Escaneie para ver nosso catálogo"
