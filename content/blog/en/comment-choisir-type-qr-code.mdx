## Why the QR Code Type Matters

All QR codes look the same visually, but they don't do the same thing. A WiFi QR code automatically connects to a network. A vCard QR code adds a contact. A URL QR code opens a web page. Choosing the right type from the start saves you from having to redo everything.

## QR Code Types Explained

### URL QR Code

**What it does**: opens a web page in the browser.

**When to use it**:
- Link to your website, a product page, or a form
- Marketing campaigns with scan tracking
- Any content accessible via a URL

**Advantage**: dynamic — you can change the destination without modifying the QR code.

### Text QR Code

**What it does**: displays plain text on the phone screen.

**When to use it**:
- Short messages (promo codes, instructions, passwords)
- Information that doesn't need to be online
- Educational treasure hunts

**Advantage**: works offline, no internet connection required.

### WiFi QR Code

**What it does**: automatically connects the phone to a WiFi network.

**When to use it**:
- Hotels, restaurants, cafes, coworking spaces
- Offices and meeting rooms
- Events and conferences

**Advantage**: no need to dictate the password — one scan and you're connected.

### vCard QR Code

**What it does**: adds a full contact (name, phone, email, company, address) to the phone's address book.

**When to use it**:
- Professional business cards
- Email signatures
- Trade show badges

**Advantage**: all your contact details saved in one scan, with zero data entry errors.

### Event QR Code

**What it does**: adds an event to the phone's calendar with date, time, location, and description.

**When to use it**:
- Invitations (weddings, birthdays, conferences)
- Event posters
- Confirmation emails

**Advantage**: reduces no-show rates — the event appears in the calendar with reminders.

### Email QR Code

**What it does**: opens the email app with the address, subject, and body pre-filled.

**When to use it**:
- Customer support (pre-filled email with context)
- Simplified contact forms
- Standardized quote requests

**Advantage**: reduces friction — the customer only has to hit "Send."

### Phone QR Code

**What it does**: initiates a call to the encoded number.

**When to use it**:
- Customer support number
- Emergency contact
- Quick business card (just the number)

### SMS QR Code

**What it does**: opens the SMS app with the number and message pre-filled.

**When to use it**:
- SMS opt-in (marketing sign-up)
- Quick orders
- Reporting or feedback

### Social Media QR Code

**What it does**: redirects to your Instagram, TikTok, LinkedIn, YouTube profile, etc.

**When to use it**:
- In-store to gain followers
- On product packaging
- On business cards

### PDF QR Code

**What it does**: opens or downloads a PDF document.

**When to use it**:
- Restaurant menus
- Brochures and catalogs
- Technical documents and user manuals
- Certificates and diplomas

### Location QR Code

**What it does**: opens the navigation app to GPS coordinates.

**When to use it**:
- Invitations with directions
- Tourist signage
- Real estate listings
