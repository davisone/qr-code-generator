## Pourquoi utiliser un QR code WiFi ?

Partager un mot de passe WiFi reste l'un des moments les plus frustrants de la vie quotidienne. Vous l'avez vécu : épeler un mot de passe de 20 caractères à un invité, un client de restaurant ou un participant d'événement. Le QR code WiFi élimine ce problème en une seule étape.

Quand une personne scanne votre QR code WiFi, son téléphone se connecte **automatiquement** au réseau. Pas de saisie manuelle, pas d'erreur de frappe, pas de confusion entre majuscules et minuscules.

**Les cas d'usage les plus fréquents :**

- Restaurants et cafés qui offrent le WiFi gratuit
- Hôtels et Airbnb pour l'accueil des clients
- Bureaux et espaces de coworking
- Événements et conférences
- Commerces et salons d'attente

## Comment générer votre QR code WiFi

### Étape 1 : Rassemblez vos informations réseau

Vous aurez besoin de trois éléments :

- **Le nom du réseau (SSID)** : exactement comme il apparaît sur votre routeur. Attention aux majuscules et aux espaces.
- **Le type de sécurité** : WPA/WPA2 (le plus courant), WEP (ancien), ou Ouvert (sans mot de passe).
- **Le mot de passe WiFi** : tel quel, sensible à la casse.

<InfoBox type="tip">
Pour trouver votre SSID et type de sécurité, ouvrez les paramètres WiFi de votre téléphone et regardez les détails de votre réseau connecté.
</InfoBox>

### Étape 2 : Générez le QR code

Rendez-vous sur le [générateur WiFi useqraft](/qr-code-generator/wifi) et remplissez les champs :

1. Entrez le nom de votre réseau (SSID)
2. Sélectionnez le type de sécurité
3. Tapez votre mot de passe
4. Cochez "Réseau caché" si applicable

Le QR code se génère en temps réel dans la zone de prévisualisation.

### Étape 3 : Téléchargez et imprimez

Choisissez le format qui convient :

- **PNG** : idéal pour le web et les impressions classiques
- **SVG** : vectoriel, parfait pour agrandir sans perte de qualité
- **PDF** : prêt à imprimer directement

### Étape 4 : Placez-le stratégiquement

L'emplacement fait toute la différence :

- **Restaurant** : sur chaque table, à côté du menu
- **Hôtel** : à la réception et dans chaque chambre
- **Bureau** : dans la salle de réunion et l'espace commun
- **Événement** : à l'entrée et sur les badges

## Le mot de passe est-il en sécurité ?

C'est la question numéro 1. La réponse : **le mot de passe est encodé directement dans l'image du QR code**. Il n'est envoyé à aucun serveur. Tout se passe localement, entre le téléphone du visiteur et votre routeur WiFi.

## Compatibilité : iPhone et Android

Bonne nouvelle : les QR codes WiFi sont supportés nativement.

- **iOS 11+** : ouvrez l'appareil photo, visez le QR code, tapez la notification
- **Android 10+** : même processus via l'appareil photo natif
- **Anciens appareils** : n'importe quelle application de scan QR fonctionne

## Conseils pour un QR code WiFi efficace

1. **Imprimez grand** : minimum 3 cm × 3 cm pour un scan fiable
2. **Bon contraste** : QR code sombre sur fond clair
3. **Testez avant d'imprimer** : scannez toujours votre QR code avec un autre téléphone
4. **Mettez à jour si vous changez de mot de passe** : un QR code statique ne peut pas être modifié après impression
5. **Ajoutez un texte d'accompagnement** : "Scannez pour vous connecter au WiFi" aide les personnes moins familières avec les QR codes
