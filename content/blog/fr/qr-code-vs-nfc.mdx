## Deux technologies, un même objectif

QR codes et NFC (Near Field Communication) permettent tous deux de transmettre des informations sans contact. Mais leurs modes de fonctionnement, coûts et cas d'usage diffèrent radicalement.

## Comment fonctionnent-ils ?

### QR code

Un motif visuel encodant des données, lu par la caméra d'un smartphone. Le téléphone décode le motif et exécute l'action (ouvrir une URL, se connecter au WiFi, ajouter un contact).

**Portée** : de quelques centimètres à plusieurs mètres selon la taille du QR code.

### NFC

Une puce électronique qui communique par radiofréquence avec un smartphone approché à moins de 4 cm. Le téléphone détecte la puce et lit les données stockées.

**Portée** : 1 à 4 cm maximum.

## Avantages du QR code

### Coût imbattable
Un QR code ne coûte rien à produire. Imprimé sur papier, affiché sur écran, gravé sur métal — le support est le seul coût. Pour une campagne marketing avec 10 000 flyers, la différence est massive.

### Portée visuelle
Un QR code sur une affiche peut être scanné depuis le trottoir d'en face. Un tag NFC nécessite de coller son téléphone dessus. Pour la signalétique, les vitrines et les affiches, le QR code gagne.

### Universalité
100% des smartphones fabriqués depuis 2018 lisent les QR codes nativement via la caméra. Le NFC nécessite une puce compatible (absente sur certains modèles anciens ou entrée de gamme).

### Utilisation de masse
Lors d'un événement, 200 personnes peuvent scanner un QR code affiché simultanément. Un tag NFC ne traite qu'une personne à la fois.

## Avantages du NFC

### Expérience fluide
Approcher son téléphone est plus rapide que d'ouvrir la caméra et de viser un QR code. Pour les interactions répétées (badge d'accès, paiement), le NFC est imbattable.

### Durabilité
Une puce NFC encapsulée résiste à l'eau, aux UV, à l'usure. Un QR code imprimé s'abîme avec le temps.

### Discrétion
Le NFC est invisible — parfait pour des produits haut de gamme où un QR code visible serait inesthétique.

### Fonctionnement sans lumière
Le NFC fonctionne dans l'obscurité totale. Le QR code a besoin de lumière (ou du flash du téléphone).

## Quand utiliser quoi ?

### QR code — idéal pour :
- **Marketing** : flyers, affiches, packaging, cartes de visite
- **Restauration** : menus, commandes, paiements
- **Événements** : billets, programme, WiFi
- **Éducation** : supports de cours, liens vers ressources
- **Immobilier** : panneaux, fiches de biens

### NFC — idéal pour :
- **Contrôle d'accès** : badges, serrures connectées
- **Paiement sans contact** : terminaux de paiement
- **Produits de luxe** : authentification, expérience premium
- **Logistique** : suivi d'inventaire, traçabilité
- **Automobile** : partage de clés numériques

### Les deux ensemble
La combinaison QR + NFC est souvent la meilleure approche. Exemples :
- **Carte de visite** : QR code au recto + puce NFC intégrée
- **Stand salon** : QR code visible à distance + tag NFC pour les visiteurs proches
- **Produit** : NFC dans le packaging + QR code sur l'étiquette

<InfoBox type="info">
Pour la grande majorité des cas d'usage marketing et communication, le QR code est le choix le plus pragmatique : gratuit, universel, personnalisable et trackable. Le NFC excelle dans les interactions physiques répétées (accès, paiement) où la rapidité prime.
</InfoBox>
