## Le QR code : standard de paiement crypto

Les QR codes sont la méthode standard pour initier un paiement en cryptomonnaie. Quand un client scanne votre QR code crypto, son application wallet pré-remplit automatiquement :

- **L'adresse du destinataire** (votre wallet)
- **Le montant** (optionnel)
- **La devise** (BTC, ETH ou LTC)

C'est le même principe que pour les virements bancaires, mais en décentralisé et instantané.

## Cryptomonnaies supportées

Le générateur QRaft supporte les trois principales :

- **Bitcoin (BTC)** : la plus utilisée, format BIP21 (`bitcoin:adresse?amount=0.01`)
- **Ethereum (ETH)** : pour les paiements et smart contracts (`ethereum:adresse?value=1e18`)
- **Litecoin (LTC)** : alternative rapide et économique (`litecoin:adresse?amount=0.5`)

## Créer votre QR code crypto

### 1. Récupérez votre adresse wallet

Ouvrez votre application wallet (Ledger Live, MetaMask, Trust Wallet, etc.) et copiez votre adresse de réception.

<InfoBox type="warning">
Vérifiez TOUJOURS l'adresse deux fois. Les transactions blockchain sont irréversibles. Une erreur d'adresse = des fonds perdus définitivement.
</InfoBox>

### 2. Générez le QR code

Rendez-vous sur le [générateur Crypto QRaft](/qr-code-generator/crypto) :

1. Sélectionnez la cryptomonnaie (BTC, ETH ou LTC)
2. Collez votre adresse wallet
3. Optionnel : définissez un montant
4. Téléchargez le QR code

### 3. Affichez-le

- **Site web** : page de donation ou de paiement
- **Point de vente** : écran ou impression au comptoir
- **Facture** : intégrez le QR code dans vos factures
- **Livestream** : affichez-le à l'écran pour recevoir des tips

## Cas d'usage concrets

### Donations
Créateurs de contenu, associations, projets open-source — affichez votre QR code crypto pour recevoir des donations sans intermédiaire.

### E-commerce
Proposez le paiement crypto comme alternative aux cartes bancaires. Le QR code simplifie l'étape de paiement.

### Point de vente physique
Affichez un QR code au comptoir avec le montant pré-rempli pour des paiements rapides.

## Sécurité

Le QR code ne contient que votre **adresse publique**. Il ne contient jamais :
- Vos clés privées
- Votre seed phrase
- Aucune information sensible

Il est parfaitement sûr de partager et d'afficher publiquement un QR code crypto.
