## Pourquoi le type de QR code compte

Tous les QR codes se ressemblent visuellement, mais ils ne font pas la même chose. Un QR code WiFi connecte automatiquement au réseau. Un QR code vCard ajoute un contact. Un QR code URL ouvre une page web. Choisir le bon type dès le départ vous évite de devoir tout refaire.

## Les types de QR codes expliqués

### QR code URL

**Ce qu'il fait** : ouvre une page web dans le navigateur.

**Quand l'utiliser** :
- Renvoyer vers votre site web, une page produit, un formulaire
- Campagnes marketing avec tracking des scans
- Tout contenu accessible via une URL

**Avantage** : dynamique — vous pouvez changer la destination sans modifier le QR code.

### QR code texte

**Ce qu'il fait** : affiche un texte brut sur l'écran du téléphone.

**Quand l'utiliser** :
- Messages courts (codes promo, instructions, mots de passe)
- Informations qui n'ont pas besoin d'être en ligne
- Chasses au trésor éducatives

**Avantage** : fonctionne hors ligne, aucune connexion internet nécessaire.

### QR code WiFi

**Ce qu'il fait** : connecte automatiquement le téléphone au réseau WiFi.

**Quand l'utiliser** :
- Hôtels, restaurants, cafés, espaces de coworking
- Bureaux et salles de réunion
- Événements et conférences

**Avantage** : plus besoin de dicter le mot de passe — un scan et c'est connecté.

### QR code vCard

**Ce qu'il fait** : ajoute un contact complet (nom, téléphone, email, entreprise, adresse) au répertoire du téléphone.

**Quand l'utiliser** :
- Cartes de visite professionnelles
- Signatures d'email
- Badges de salon

**Avantage** : toutes vos coordonnées enregistrées en un scan, sans erreur de saisie.

### QR code événement

**Ce qu'il fait** : ajoute un événement au calendrier du téléphone avec date, heure, lieu et description.

**Quand l'utiliser** :
- Invitations (mariage, anniversaire, conférence)
- Affiches d'événements
- Emails de confirmation

**Avantage** : réduit le taux de no-show — l'événement apparaît dans le calendrier avec rappel.

### QR code email

**Ce qu'il fait** : ouvre l'application email avec l'adresse, le sujet et le corps pré-remplis.

**Quand l'utiliser** :
- Support client (email pré-rempli avec le contexte)
- Formulaires de contact simplifié
- Demandes de devis standardisées

**Avantage** : réduit la friction — le client n'a qu'à cliquer "Envoyer".

### QR code téléphone

**Ce qu'il fait** : lance un appel vers le numéro encodé.

**Quand l'utiliser** :
- Numéro de support client
- Contact d'urgence
- Carte de visite rapide (juste le numéro)

### QR code SMS

**Ce qu'il fait** : ouvre l'application SMS avec le numéro et le message pré-remplis.

**Quand l'utiliser** :
- Inscription par SMS (opt-in marketing)
- Commandes rapides
- Signalement ou feedback

### QR code réseaux sociaux

**Ce qu'il fait** : redirige vers votre profil Instagram, TikTok, LinkedIn, YouTube, etc.

**Quand l'utiliser** :
- En boutique pour gagner des followers
- Sur les emballages produits
- Sur les cartes de visite

### QR code PDF

**Ce qu'il fait** : ouvre ou télécharge un document PDF.

**Quand l'utiliser** :
- Menus de restaurant
- Brochures et catalogues
- Documents techniques et modes d'emploi
- Certificats et diplômes

### QR code localisation

**Ce qu'il fait** : ouvre l'application de navigation vers des coordonnées GPS.

**Quand l'utiliser** :
- Invitations avec itinéraire
- Signalétique touristique
- Annonces immobilières

<InfoBox type="tip">
En cas de doute, le QR code URL est le plus polyvalent. Il fonctionne pour tout contenu accessible via un lien, il est dynamique (modifiable) et il permet le tracking des scans. Les autres types sont plus spécialisés mais offrent une meilleure expérience utilisateur pour leur cas d'usage spécifique.
