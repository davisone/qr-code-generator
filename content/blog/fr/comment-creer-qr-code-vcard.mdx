## Le problème avec les cartes de visite traditionnelles

Vous distribuez votre carte de visite lors d'un salon professionnel. Le contact la glisse dans sa poche, rentre chez lui, et... la carte finit dans un tiroir. Ou pire, à la poubelle. Selon une étude, **88% des cartes de visite sont jetées dans la semaine**.

Le QR code vCard change cette dynamique. Quand quelqu'un scanne votre QR code, vos coordonnées sont **sauvegardées directement dans les contacts** de son téléphone. Pas de saisie manuelle, pas de risque de perte.

## Qu'est-ce qu'un QR code vCard ?

Un QR code vCard encode vos informations de contact dans un format standardisé (vCard/VCF). Il peut contenir :

- Nom complet
- Numéro de téléphone
- Adresse email
- Entreprise et poste
- Site web
- Adresse postale
- Notes personnalisées

Quand le code est scanné, le téléphone propose d'ajouter le contact avec toutes ces informations pré-remplies.

## Créer votre QR code vCard en 3 minutes

### Étape 1 : Préparez vos informations

Décidez quels champs inclure. Le minimum recommandé :

- **Nom complet** (obligatoire)
- **Téléphone** (le plus utilisé)
- **Email** (essentiel pour le business)

<InfoBox type="tip">
Moins c'est plus. Un QR code avec trop de données devient complexe et plus difficile à scanner. Limitez-vous aux 5-6 champs essentiels.
</InfoBox>

### Étape 2 : Générez le QR code

Rendez-vous sur le [générateur vCard useqraft](/qr-code-generator/vcard) :

1. Entrez votre nom complet

### Étape 3 : Intégrez-le à vos supports

Les meilleurs emplacements pour votre QR code vCard :

- **Carte de visite** : au verso, avec un texte "Scannez pour sauvegarder mes coordonnées"
- **Signature email** : en image dans votre signature professionnelle
- **Badge événement** : imprimé sur votre badge de conférence
- **Présentation** : sur la dernière slide de vos présentations
- **CV** : en en-tête pour faciliter la prise de contact

## Questions fréquentes

**Le QR code vCard fonctionne-t-il sur tous les téléphones ?**

Oui. Tous les smartphones modernes (iOS et Android) supportent le format vCard nativement via l'application appareil photo.

**Puis-je inclure ma photo ?**

Le format vCard supporte les photos, mais cela alourdit considérablement le QR code et rend le scan moins fiable. Privilégiez un QR code léger et ajoutez votre photo via votre profil LinkedIn ou votre site web.

**Que se passe-t-il si je change de numéro ?**

Vous devrez générer un nouveau QR code et remplacer l'ancien sur vos supports. Avec un compte Pro useqraft, vous pouvez utiliser un QR code dynamique modifiable à tout moment.
