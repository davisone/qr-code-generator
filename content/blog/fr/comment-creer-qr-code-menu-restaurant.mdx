## Pourquoi digitaliser votre menu avec un QR code ?

Depuis la pandémie, les menus QR code sont devenus la norme dans la restauration. Mais au-delà de l'aspect sanitaire, ils offrent des avantages concrets :

- **Économie** : plus besoin de réimprimer à chaque changement de prix ou de plat
- **Hygiène** : zéro contact, zéro passage de main en main
- **Flexibilité** : modifiez votre menu en temps réel (plat du jour, rupture de stock)
- **Écologie** : réduction du papier et de l'encre

## Prérequis : hébergez votre menu en ligne

Avant de créer le QR code, vous avez besoin d'une URL pointant vers votre menu. Trois options :

### Option 1 : Page sur votre site web
Si vous avez déjà un site, créez une page dédiée `/menu` avec vos plats, prix et photos. C'est la meilleure option pour le SEO.

### Option 2 : PDF en ligne
Créez votre menu en PDF et hébergez-le sur Google Drive (partage public) ou directement sur votre site. Simple et rapide.

### Option 3 : Plateforme de menu digital
Des services comme Carte.co, Menu.app ou même une simple page Notion publique peuvent servir de menu en ligne.

<InfoBox type="tip">
Quelle que soit l'option choisie, assurez-vous que la page est **responsive** (lisible sur mobile) et se charge rapidement. Vos clients scanneront depuis leur téléphone.
</InfoBox>

## Où placer votre QR code menu

L'emplacement est crucial pour l'adoption :

- **Sur chaque table** : autocollant ou support en plexiglas
- **À l'entrée** : pour que les clients consultent le menu en attendant
- **Sur la vitrine** : les passants peuvent consulter votre carte
- **Sur les emballages takeaway** : fidélisez vos clients à emporter
- **Sur Google Maps** : ajoutez le lien du menu à votre fiche Google Business

## Conseils pour maximiser l'utilisation

1. **Ajoutez un texte d'accompagnement** : "Scannez pour voir notre menu" avec une petite flèche
2. **Gardez le même URL** : si vous changez le contenu de la page (et pas l'URL), votre QR code reste valide
3. **Testez régulièrement** : vérifiez que le lien fonctionne toujours
4. **Proposez une alternative** : gardez quelques menus papier pour les clients qui préfèrent

## Menu statique vs dynamique

Avec le QR code gratuit de QRaft, le lien est **statique** — il pointe toujours vers la même URL. Si votre menu est une page web que vous mettez à jour, c'est parfait : le QR code ne change pas, mais le contenu évolue.

Pour un QR code dont vous pouvez **changer la destination** sans réimprimer (rediriger vers un nouveau PDF par exemple), un compte Pro avec QR codes dynamiques est recommandé.
